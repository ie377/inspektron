## 2.3. РУКОВОДСТВО ПО ЭКСПЛУАТАЦИИ (Извлечение)

### Назначение
Анализатор Инспектрон-100 предназначен для непрерывного измерения CO2, CO и PM.

### Технические характеристики

| Параметр | Значение |
|----------|----------|
| Диапазон CO2 | 400-5000 ppm |
| Диапазон CO | 0-100 ppm |
| Диапазон PM | 0-1000 ug/m3 |
| Погрешность CO2 | +-2% |
| Погрешность CO | +-2% |
| Погрешность PM | +-5% |
| Выходной сигнал | 4-20mA, RS-485 |
| Питание | 12-24V DC |
| Мощность | <=2W |
| Габариты | 180x120x65 mm |
| Масса | 0.8 kg |
| IP | IP54 |
| Температура | -20...+50C |

### Схема подключения

    +24V ----+----[+] БП
             |
            GND ---[–] БП
                     |
          +----------+----------+
          |   Инспектрон-100     |
          |                     |
          | 4-20mA ----> SCADA   |
          | RS-485 --> Modbus   |
          +---------------------+

### Калибровка
Проводится 1 раз в 12 месяцев. Подключить к ПК через USB-UART, запустить ПО "Inspektron Calibrator".

### Требования безопасности
- Не эксплуатировать при T > +50C
- Не допускать попадания влаги
- Подключать при обесточенном оборудовании
- Заземление обязательно
