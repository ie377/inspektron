# 1. ОБЩИЕ СВЕДЕНИЯ

## 1.1. Назначение и область применения

Анализаторы воздуха серии **Инспектрон** предназначены для измерения концентрации газов и взвешенных частиц в воздухе атмосферы. Приборы применяются в следующих отраслях:

- Промышленная вентиляция и кондиционирование
- Экологический мониторинг
- Контроль качества воздуха в учреждениях здравоохранения
- Санитарно-эпидемиологический контроль
- Научные исследования

## 2.3. РУКОВОДСТВО ПО ЭКСПЛУАТАЦИИ

**Инспектрон-100. Издание 2026**

\bigskip

### 2.3.1. Назначение

Анализатор Инспектрон-100 предназначен для непрерывного измерения концентрации углекислого газа (CO₂), оксида углерода (CO) и взвешенных частиц PM1, PM2.5, PM10 в воздухе атмосферы.

### 2.3.2. Технические характеристики

| Параметр | Значение |
|----------|----------|
| Диапазон измерения CO₂ | 400–5000 ppm |
| Диапазон измерения CO | 0–100 ppm |
| Диапазон измерения PM | 0–1000 мкг/м³ |
| Погрешность CO₂ | ±2% |
| Погрешность CO | ±2% |
| Погрешность PM | ±5% |
| Выходной сигнал | 4–20 мА, RS-485 (Modbus RTU) |
| Напряжение питания | 12–24 В DC |
| Потребляемая мощность | ≤2 Вт |
| Габариты | 180×120×65 мм |
| Масса | 0,8 кг |
| Класс защиты | IP54 |
| Рабочая температура | −20…+50°C |

### 2.3.3. Устройство и принцип работы

Прибор состоит из следующих функциональных блоков:

1. **Датчик CO₂** — NDIR-сенсор (недисперсионный инфракрасный)
2. **Датчик CO** — электрохимический сенсор
3. **Датчик PM** — лазерный рассеивающий сенсор
4. **Микроконтроллер** — обработка сигналов
5. **Интерфейсный модуль** — выход 4–20 мА, RS-485

Принцип работы основан на поглощении инфракрасного излучения молекулами газа.

### 2.3.4. Схема подключения

```
     +24V  ────────┬────────────────┬───────[+] БП
                   │                │
                  GND ───────┬───────[–] БП
                           │
          ┌─────────────────┼─────────────────┐
          │         Инспектрон-100              │
          │                                   │
          │  4-20mA  ────────→  Контроллер     │
          │                 (SCADA)            │
          │                                   │
          │  RS-485 A ───────→  RS-485        │
          │  RS-485 B ───────→  (Modbus)      │
          │                                   │
          └───────────────────────────────────┘
```

### 2.3.5. Калибровка

Калибровка проводится **1 раз в 12 месяцев** или при отклонении показаний более чем на ±3%.

**Порядок калибровки:**

1. Подключить прибор к ПК через USB-UART адаптер
2. Запустить ПО «Inspektron Calibrator»
3. Выбрать режим «Калибровка CO₂ (400 ppm)»
4. Подать эталонный газ 400 ppm
5. Дождаться стабилизации показаний
6. Подтвердить калибровку

### 2.3.6. Требования безопасности

- Не эксплуатировать при температуре выше +50°C
- Не допускать попадания влаги внутрь корпуса
- Подключение производить при обесточенном оборудовании
- Заземление корпуса обязательно для класса защиты IP54

\bigskip

### 3.3.1. Назначение

Анализатор Инспектрон-200 предназначен для измерения концентрации летучих органических соединений (TVOC), формальдегида (HCHO), углекислого газа (CO₂) и взвешенных частиц PM2.5, PM10 в воздухе помещений.

### 3.3.2. Технические характеристики

| Параметр | Значение |
|----------|----------|
| Диапазон измерения CO₂ | 400–5000 ppm |
| Диапазон измерения TVOC | 0–10 ppm |
| Диапазон измерения HCHO | 0–1 мг/м³ (0–1 ppm) |
| Диапазон измерения PM2.5/PM10 | 0–1000 мкг/м³ |
| Погрешность CO₂ | ±2% |
| Погрешность TVOC | ±5% |
| Погрешность HCHO | ±5% |
| Погрешность PM | ±5% |
| Выходной сигнал | 4–20 мА, RS-485, 0–10 В |
| Напряжение питания | 12–24 В DC |
| Потребляемая мощность | ≤3 Вт |

| Масса | 1,8 кг |
| Класс защиты | IP54 |
| Рабочая температура | −10…+50°C |

### 3.3.3. Устройство и принцип работы

Прибор оснащён следующими сенсорами:

1. **TVOC** — PID-сенсор (фотоионизационный)
2. **HCHO** — электрохимический сенсор
3. **CO₂** — NDIR-сенсор
4. **PM** — лазерный сенсор рассеивания
