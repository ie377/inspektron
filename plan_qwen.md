- **История:** Текстовый таймлайн (год основания, запуск линии, выход на ГОСТ, масштабирование)
- **Производство:** Описание цеха, этапы сборки, контроль качества (столы поверки, эталонные газы, климатические камеры)

- **Гарантия и SLA:** 24 мес, замена/ремонт ≤14 дней, техподдержка 9:00–18:00 МСК
- **Документация:** Архив PDF, инструкции по интеграции (Modbus RTU, OPC-UA)
- **FAQ:** 8–10 вопросов: `Как подключить к контроллеру?`, `Можно ли использовать во взрывоопасной зоне?`, `Как часто менять сенсоры?`

### 📞 Контакты
- Форма запроса (дублирует главную, но с полем `Приоритет срочности`)
- Реквизиты (ИНН, ОГРН, р/с, юр. адрес)

## 🔹 4. Техническая реализация

| Аспект | Решение |
|--------|---------|
| **CMS** | **Tilda** (Zero Block + стандартные блоки) → быстро, дешево, достаточно для 5–7 страниц. Альтернатива: WordPress + ACF + GeneratePress (если планируется блог/кабинет дилеров) |
| **Формы** | Встроенные Tilda / WPForms → интеграция с Битрикс24 или AmoCRM через Webhook. Защита: reCAPTCHA v3 + honeypot |
| **Аналитика** | Яндекс.Метрика (цели: `submit_form`, `click_download_pdf`, `click_phone`), коллтрекинг (Zvonok/Calltouch), UTM-метки для рекламы |
| **Производительность** | Сжатие изображений → WebP/SVG, отложенная загрузка PDF, минификация CSS/JS, кэширование, CDN (если РФ-хостинг: Timeweb/Beget) |
| **SEO-техника** | ЧПУ `/catalog/detectron-200`, мета-теги по шаблону `Title: Анализатор CO2 и TVOC Детектрон-200 | Производство РФ`, `H1` один на страницу, `schema.org` (`Product`, `Organization`, `FAQPage`), `sitemap.xml`, `robots.txt` без блокировки CSS/JS |
| **Безопасность** | HTTPS, скрытие версий CMS, ограничение попыток входа, регулярные бэкапы |

### 📈 Конверсионная механика (B2B)
- **Нет корзины** → только `Запросить КП` и `Скачать спецификацию`
- **Лид-магниты**: `Таблица сравнения анализаторов (Excel)`, `Чек-лист выбора датчиков для СЗЗ`, `Протокол типовой поверки (образец)`
- **Доверие**: сканы сертификатов, номер в реестре ФГИС «АРШИН», указание завода-изготовителя, гарантия 24 мес, SLA в цифрах
- **Ретаргетинг**: пиксель ВК/Яндекс на посетителей карточек >15 сек или скачавших PDF
- **Email-воронка**: после отправки формы → автописьмо с PDF + предложение консультации → напоминание через 3/7 дней

## 🔹 6. Чек-лист запуска

| Этап | Задачи | Срок |
|------|--------|------|
| 1. Подготовка | Сбор ТТ, спецификаций, сканов сертификатов, текстов, SVG-иконок | 3–5 дн. |
| 2. Прототип | Wireframe в Figma (мобильный + десктоп), утверждение таблиц и CTA | 3 дн. |
| 3. Верстка/CMS | Настройка Tilda/WordPress, Zero-блоки, таблицы, формы, интеграция CRM | 5–7 дн. |
| 4. Контент | Загрузка текстов, PDF, мета-теги, schema, проверка адаптива | 2–3 дн. |
| 5. Тесты | Формы → CRM, скорость (PageSpeed >85), мобильный скролл таблиц, битые ссылки, reCAPTCHA | 2 дн. |
| 6. Запуск | Подключение домена, SSL, индексация в Яндекс.Вебмастер, цели Метрики, резервное копирование | 1 дн. |
| 7. Post-launch | A/B тест CTA (`Запросить КП` vs `Скачать КП`), добавление 3 статей в раздел `База знаний`, настройка ретаргетинга | Постоянно |
